¿Qué es el Mainframe de IBM Z?

Es un conjunto de Sistemas de Computación o Servidores, utilizado para procesar transacciones a gran escala y diseñado para ejecutar aplicaciones críticas con las siguientes características:

  1. Alta seguridad, ofreciendo cifrado de datos en tiempo real a nivel de hardware.
  2. Escalabilidad y rendimiento, procesando miles de millones de transacciones por día y siendo capaz de ejecutar simultáneamente miles de aplicaciones bajo diferentes cargas de trabajo, por medio de una arquitectura que permite escalar horizontal y verticalmente sin pérdida de rendimiento. Tipos de cargas de trabajo: OLTP o Batch.
  3. Alta disponibilidad, funcionando con una disponibilidad del 99.999% y soportando operaciones ininterrumpidas, incluso durante mantenimiento y actualizaciones.
  4. Virtualización y contenedores, ejecutando varios Sistemas Operativos (S.O) al mismo tiempo. Es compatible con Linux, Kubernetes y Docker. Ésta última es una plataforma moderna para Cloud híbrida y DevOps.

Su uso se da en:

  1. Bancos, para procesar millones de transacciones diarias.
  2. Aerolíneas, para gestionar reservas y sistemas de embarque en tiempo real.
  3. Gobiernos, para gestionar los sistemas de la administración.
  4. Ventas, comercio y logística, para llevar a cabo la trazabilidad y la facturación a gran escala.

Está desarrollado por la empresa estadounidense de IBM.

No obstante, aunque se asocie a IBM como sistemas clásicos o una marca antigua, el Mainframe de IBM Z es una plataforma moderna, segura, ultraescalable y con soporte para: Inteligencia Artificial (IA) integrada, contenedores, microservicios, desarrollo de operaciones e integración con APIs REST y sistemas modernos. Además, está diseñada para ejecutar cargas de trabajo críticas en organizaciones que no pueden permitirse fallos, pérdidas de datos ni tiempos de espera.

Sistema o Servidor Mainframe de IBM Z

Es un Servidor de gama alta especializado en procesamiento masivo de datos, donde el primero fue el zEC12 (2012) y a partir de éste, han habido mejoras, evolucionando así en los siguientes:

  1. z13 (2015), centrado en analítica en tiempo real y movilidad.
  2. z14 (2017), incluye cifrado total de datos o pervasive encryption.
  3. z15 (2019), enfocado en nube híbrida y privacidad de datos.
  4. z16 (2022), es el primer Mainframe con IA integrada acelerada por hardware (Telum chip).
  5. z17 (2024), orientado a IA generativa, aceleración cuántica y seguridad Zero Trust.

Su Sistema Operativo (S.O) es el z/OS, usado a través de VPN.
